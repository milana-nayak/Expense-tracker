# 💰 Expense Tracker

A modern and responsive Expense Tracker web application built using **HTML, CSS, and JavaScript**. It helps users manage their daily expenses with an easy-to-use interface and stores data using **Local Storage**, so expenses remain saved even after refreshing the page.

---

## 🚀 Live Demo

🔗 https://milana-nayak.github.io/Expense-tracker/

---

## ✨ Features

- ➕ Add new expenses
- ✏️ Edit existing expenses
- 🗑️ Delete expenses
- 🔍 Search expenses by name
- 📂 Filter expenses by category
- 💰 Dashboard showing:
  - Total Expense
  - Total Transactions
  - Highest Expense
- 💾 Local Storage support
- 📱 Responsive design for mobile and desktop
- 🎨 Clean and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ⚙️ How to Run

1. Clone the repository

```bash

```

2. Open the project folder.

3. Open `index.html` in your browser.
