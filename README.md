# Expense Tracker Web App

A dynamic web application to track daily expenses with real-time updates and data persistence.

## Features
- Add new expenses
- Delete expenses
- Auto calculation of total amount
- Data stored using localStorage (persists after refresh)

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation, Events, localStorage)

## Live Demo
https://milana-nayak.github.io/Expense-Tracker/

## Key Concepts
- DOM Manipulation
- Event Handling
- State Management using arrays
- Data persistence using localStorage
